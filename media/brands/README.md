## Naming rules

Use lowercase, hyphenated filenames:

```text
logo.png
logo-white.png
brand-mark.png
collection-logo.png
icon-feature-name.svg
```

## Public URL format

Use jsDelivr for product descriptions:

```text
https://cdn.jsdelivr.net/gh/jacobsherman-spec/discount-furniture-site-scripts@main/media/brands/BRAND-FOLDER/logo.png
```

Add `?v=1` to refresh cached images after replacing files.
