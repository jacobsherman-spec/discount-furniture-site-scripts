# Discount Furniture Brand Templates

This folder is the working home for product-description templates used in Lightspeed / Ecwid product descriptions.

## Current plan

Use this folder as the master registry for future product-description work.

Recommended structure:

```text
brand-templates/
  README.md
  registry/
    brand-template-registry.html
  brands/
    liberty.html
    a-america.html
    steve-silver.html
    best-home-furnishings.html
    la-z-boy.html
    jackson-catnapper.html
    spring-air.html
    bernards.html
```

## Rules

- Keep product-description templates brand-specific.
- Keep global PDP CSS focused on the product-page shell only.
- Do not force one brand style across every product description.
- Move brand logos/icons from Dropbox to `media/brands/` over time.
- Use GitHub as the master copy, then paste final HTML into Lightspeed.

## Styling direction

Main Discount Furniture brand:

- Navy: `#003349`
- Warm neutral: `#E6E2D5`
- Headings: Georgia
- Body: Montserrat

Brand templates may still use manufacturer-specific colors where appropriate.
